# 웹 콘텐츠 AI 요약기

원클릭으로 웹페이지 콘텐츠를 요약하고 스트리밍 Markdown 형식으로 출력합니다. 커스텀 LLM API 설정을 지원합니다 (OpenAI 및 모든 호환 API).

## 기능

- **원클릭 요약** — 도구 모음 아이콘, 우클릭 메뉴, 또는 `Ctrl+Shift+S` (Mac: `Cmd+Shift+S`) 단축키
- **스트리밍 출력** — AI 생성 콘텐츠가 실시간으로 표시되며 전체 응답을 기다릴 필요가 없음
- **Markdown 렌더링** — 서식이 적용된 Markdown으로 결과를 표시하고 원클릭 복사 지원
- **멀티모달 비전 지원** — 이미지/동영상 페이지에서 실제 이미지를 비전 지원 모델(GPT-4o, Claude, Gemini 등)에 전송하여 텍스트 메타데이터만이 아닌 시각적 콘텐츠 분석 수행
- **콘텐츠 유형 인식** — 기사/동영상/오디오/이미지 페이지를 자동 감지하고 각 유형에 맞는 추출 및 요약 전략 적용
- **토큰 예측** — API 호출 전 토큰 소비량을 예측하고, 높은 사용량일 경우 확인 대화상자 표시
- **다국어 출력** — 페이지 언어를 따르거나 출력 언어 지정 가능 (중국어, 영어, 일본어, 한국어, 프랑스어, 독일어)
- **커스텀 API** — OpenAI 및 모든 OpenAI 호환 API 지원 (DeepSeek, Qwen, LiteLLM 등)
- **모델 검색** — API 엔드포인트에서 사용 가능한 모델을 원클릭으로 가져오기
- **연결 테스트** — 사용 전 API 설정 확인

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 열기
2. 우측 상단의 **"개발자 모드"** 활성화
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 확장 프로그램 폴더(`manifest.json`이 포함된 디렉토리) 선택
5. 브라우저 도구 모음에 확장 프로그램 아이콘이 나타납니다

## 설정 방법

### 1. API 키 발급

#### OpenAI
1. [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys) 방문
2. 회원가입/로그인 후 API 키 생성
3. 확장 프로그램 설정에서 입력:
   - API 기본 URL: `https://api.openai.com/v1`
   - API 키: `sk-...`
   - '📡 모델 조회'를 클릭하여 사용 가능한 모델 목록 확인

#### DeepSeek
1. [https://platform.deepseek.com/api_keys](https://platform.deepseek.com/api_keys) 방문
2. API 키 발급
3. 설정에서 입력:
   - API 기본 URL: `https://api.deepseek.com/v1`
   - 모델: `deepseek-chat`
   - **참고**: DeepSeek 모델은 비전/이미지 분석을 지원하지 않습니다

#### 기타 호환 제공업체
OpenAI Chat Completions API와 호환되는 모든 서비스를 사용할 수 있습니다. 이미지 분석이 필요한 경우 멀티모달 입력을 지원하는 모델(GPT-4o, GPT-4.1, Claude 3/4, Gemini 1.5/2.0 등)을 사용하세요.

### 2. 연결 테스트 (권장)

1. **"🔌 연결 테스트"** 버튼을 클릭하여 API 접근 가능 여부 확인
2. 성공 시 "✅ 연결 성공" 메시지와 함께 사용 가능한 모델 수가 표시됩니다
3. 실패 시 구체적인 오류 원인이 표시됩니다

### 3. 사용 가능한 모델 검색

1. API 기본 URL과 API 키 입력
2. **"📡 모델 조회"** 버튼 클릭
3. 드롭다운에서 모델 이름을 클릭하여 선택

## 사용 방법

1. 임의의 웹페이지 열기 (블로그 글, 뉴스 기사, 동영상 페이지, 이미지 갤러리 등)
2. **방법 A**: 확장 프로그램 아이콘 클릭 → **"🔍 요약하기"** 클릭
3. **방법 B**: 페이지에서 우클릭 → **"🔍 페이지 콘텐츠 요약"** 선택
4. **방법 C**: `Ctrl+Shift+S` (Mac: `Cmd+Shift+S`) 단축키
5. 요약이 **실시간 스트리밍**으로 표시됩니다
6. **"📋 Markdown 복사"**를 클릭하여 텍스트 복사
7. 생성 중 **"✕ 취소"**를 클릭하여 중단 가능

## 콘텐츠 추출 전략

| 페이지 유형 | 추출 내용 | 요약 형식 |
|-----------|---------|---------|
| 기사/블로그 | 제목, 설명, 본문 (최대 8000자) | 제목 → 요약 → 핵심 포인트 → 결론 |
| 동영상 | 제목, 채널, 길이, 설명, **썸네일/프레임 (비전 분석용)** | 동영상 정보 → 화면 분석 → 콘텐츠 요약 → 주요 하이라이트 |
| 오디오/팟캐스트 | 제목, 제작자, 설명, **스크립트/가사/쇼 노트** | 오디오 정보 → 콘텐츠 요약 → 주요 내용 |
| 이미지 갤러리 | **이미지 URL (최대 5장, 비전 분석용)**, alt 텍스트, 페이지 컨텍스트 | 갤러리 개요 → 이미지별 상세 분석 |

**비전 모드** 자동 활성화 조건:
- 페이지 유형이 이미지, 동영상, 또는 시각적 요소가 포함된 오디오 페이지
- 설정된 모델이 멀티모달 입력을 지원하는 경우 (GPT-4o, Claude, Gemini 등)
- 모델이 비전을 지원하지 않는 경우 자동으로 텍스트 전용 모드로 폴백

## 토큰 사용량 예측

- API 호출 전 토큰 소비량 사전 예측
- 일반 사용량 (3,000 토큰 미만): 바로 실행
- 높은 사용량 (3,000 토큰 이상): 확인 대화상자 표시
- 모델 제한의 90% 초과: 경고 대화상자 표시
- 비전 모드에서 이미지 토큰도 예측에 포함 (이미지당 약 85 토큰)
- 사용자는 대화상자에서 계속 또는 취소 선택 가능

## 개인정보 보호 및 보안

- **데이터 수집 없음** — 이 확장 프로그램은 어떠한 사용자 데이터도 수집하지 않습니다
- API 키는 Chrome `storage.sync` 암호화로 로컬 저장
- API 키 입력 필드는 비밀번호 유형이며 표시/숨기기 전환 가능
- 모든 API 요청은 Background Service Worker를 통해 HTTPS로 암호화되어 전송
- 페이지 텍스트 콘텐츠는 사용자가 직접 구성한 서드파티 API 엔드포인트로만 전송
- 광고 없음, 추적 없음, 텔레메트리 없음

## 기술 스택

- Manifest V3
- 순수 HTML/CSS/JS (프레임워크 의존성 없음)
- 내장 경량 Markdown 렌더러 (의존성 없음)
- OpenAI Chat Completions API + SSE 스트리밍
- Background Service Worker가 모든 API 요청을 프록시
- `host_permissions` 불필요 (Chrome 웹 스토어 요구사항 준수)
